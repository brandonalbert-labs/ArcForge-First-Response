# Troubleshooting Flow

## Standard Flow

1. Identify the reported issue
2. Confirm the scope
3. Isolate the likely failure area
4. Test the simplest causes first
5. Apply the fix
6. Verify normal operation
7. Document the result

## Workstation Health Triage

### System

- Check hostname
- Check OS version
- Check uptime
- Check CPU and memory usage

### Storage

- Confirm free disk space
- Flag low disk space conditions

### Network

- Confirm default gateway
- Test gateway connectivity
- Test internet connectivity
- Test DNS resolution

### Software

- Check for required applications
- Confirm development/support tools are installed

### Security

- Check firewall status
- Check Defender status
- Review local administrators
