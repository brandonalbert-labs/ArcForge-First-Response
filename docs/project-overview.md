# Project Overview

## Environment

ArcForge Studio is a fictional small game studio used to frame this IT automation project.

Example workstation types:

- DEV-WKS-01: Game developer workstation
- ART-WKS-01: 3D artist workstation
- QA-WKS-01: Game testing workstation
- BUILD-SRV-01: Build server
- FILE-SRV-01: Asset storage server
- NET-FW-01: Firewall/router

## Why This Project Exists

Game studios depend on reliable workstations, stable networking, access to development tools, and fast troubleshooting.

This toolkit is designed to help IT support staff quickly collect baseline workstation health information and identify common issues before escalation.

## Skills Demonstrated

- PowerShell scripting
- Windows endpoint support
- Troubleshooting methodology
- System health checks
- Network diagnostics
- Documentation
- Git and GitHub workflow
